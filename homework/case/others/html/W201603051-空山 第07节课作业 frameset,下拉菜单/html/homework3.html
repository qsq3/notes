1.基础标签

span 标签被用来组合文档中的行内元素,以便通过样式来格式化他们。

font 标签规定文本的字体、字体尺寸、字体颜色已被淘汰

b标签和stron标签:文本为粗体格式,

i标签和em标签:文本为斜体格式,i标签还用于iconfont矢量图标

strong 标签和 em 标签一样，用于强调文本，但它强调的程度更强一些,
影响搜索引擎权重,但不要超过六个。最好使用在句子中,词语中,有利于搜索引擎抓取.
影响权重的还有h1,h2,h3,

style="font-weight: normal 去掉粗体

style="font-style: normal 去掉斜体



2.超链接a标签：

a 标签可以从通过点击从一个页面跳到另一个页面
a标签： 链接/ 下载/锚点
target 链接打开方式
blank 新窗口
self 当前窗口
base target="_blank" 定义页面链接定义新窗口打开方式,注意锚点链接也会被设置为新窗口打开


3.伪类：

伪类用于向被选中元素添加特殊的效果。
link 未访问（默认）
hover 鼠标悬停（鼠标划过）
active 链接激活（鼠标按下）
visited 访问过后（点击过后）
A四个伪类的顺序：
link visited hover active
（love hate 记忆方法！）
a伪类的应用：
a、四个伪类全用(搜索引擎、新闻门户、小说网站)
b、一般网站只用（ a{ } a:hover{ } ）
IE6不支持a以外其它任何标签的伪类；
